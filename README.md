# Medical Scheduling Applicaton  

## Tech stack
- Javascript + Typescript  
- Nodejs  
- Nestjs
- PostgreSQL under Docker (TBA)
- GraphQL (TBA)

## Install and run  
1. clone repo  
2. ```$ npm build```  
3. run with either ```$ npm run start``` or ```$ npm run dev```  
4. server listens to port 3000
